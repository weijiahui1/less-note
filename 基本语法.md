## 1.变量
> @ 符号定义变量，使用 @ 符号获取变量，仅仅将 @变量名 看成是一个字符串。

    @classname: main;
    @color: red;

    .@classname{
         background-color: @color;
    }

> 从上面例子中可以看到，变量不仅仅可以作为样式属性值：background-color: @color;，还可以作为类名：.@classname 表示的就是 .main。这也就是为什么说仅仅将 @变量名 看成是一个字符串。

## 2. 混合
> Mixins是一种将一组属性从一个规则集包含（“混入”）到另一个规则集的方法。
    
    #menu a {
      color: #111;
      border-top: dotted 1px black;
      border-bottom: solid 2px black;
    }

    .post a {
      color: red;
      border-top: dotted 1px black;
      border-bottom: solid 2px black;
    }
    改写成：
    .bordered {
      border-top: dotted 1px black;
      border-bottom: solid 2px black;
    }
    #menu a {
      color: #111;
      .bordered;
    }

    .post a {
      color: red;
      .bordered;
    }

### 3.嵌套规则
> Less使您能够使用嵌套代替或与级联结合使用。

    #header {
      color: black;
    }
    #header .navigation {
      font-size: 12px;
    }
    #header .logo {
      width: 300px;
    }
    改写成：
    #header {
      color: black;
      .navigation {
        font-size: 12px;
      }
      .logo {
        width: 300px;
      }
    }
> 生成的代码更简洁，并模仿HTML的结构。
